📝 TaskMaster: Vanilla JS To-Do List
A clean, functional, and lightweight To-Do List application. This project serves as a practical demonstration of DOM manipulation and event handling using HTML5, CSS3, and Vanilla JavaScript.


🚀 Features
Task Management: Easily add new tasks to your list in real-time.
Dynamic UI: Automatically updates the list without refreshing the page.
Minimalist Design: Focuses on a clutter-free user interface to enhance productivity.
Responsive Structure: Built with a flexible container to look great on any screen size.


🛠️ Tech Stack
HTML5: Semantic structure including an input-based task entry system.
CSS3: Styled for a modern, professional look (linked via style.css).
JavaScript (ES6): Pure "Vanilla" JS used for capturing user input and dynamically generating list items (<li>).


📁 Project Structure
Plaintext
├── index.html   # The structural skeleton
├── style.css    # Custom styling and layout
└── script.js    # Logic for adding and managing tasks
